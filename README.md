# MSCS-532-Assignment4

# Heap Data Structures - Implementation, Analysis, and Applications

## Project Overview

This project demonstrates the implementation and analysis of Heap data structures using Python. The assignment consists of two main parts.

The first part implements the Heap Sort algorithm and compares its performance with Quick Sort and Merge Sort using different input sizes and data distributions.

The second part implements a Priority Queue using a Binary Max Heap to simulate a simple task scheduling system. The program performs operations such as inserting tasks, increasing task priority, extracting the highest-priority task, and checking whether the queue is empty.

---


## Requirements

- Python 3.14.2

No additional Python libraries are required because this project only uses built-in Python modules.

---

## How to Run the Programs

### Part 1 - Heap Sort

Open VS Code and run:

```bash
py heapsort.py
```

### Part 2 - Priority Queue

Run the following command:

```bash
py priority_queue.py
```
